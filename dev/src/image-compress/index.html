画像仕分け圧縮くん — 種類別に最適化する画像圧縮ツール

画像の役割（背景・写真・イラスト・アイコン・スクショ）ごとに最適な圧縮を当てる WebP 圧縮ツール。すべてブラウザ内で処理するため、画像をサーバーに送信しません。

{{!-- 投入: ドロップゾーン ＋ ファイル選択（D&D 代替・A11y/モバイル必須） --}}

画像をここにドラッグ＆ドロップ

PNG / JPEG / WebP（一度に 30 枚・1 枚 25MB まで目安）

全部の種別を一括で選ぶ
{{!-- 自動推測トグル（Stage4 で実ロジック接続済み・初期値 ON / spec §5） --}}

大きすぎる画像は、Web 表示に十分なサイズへ自動で縮小します。

出力フォーマット（複数選べます）

選んだ形式の数だけファイルを書き出し、一括ダウンロードの ZIP にまとめます。「準備中」は今後のアップデートで使えるようになります。
